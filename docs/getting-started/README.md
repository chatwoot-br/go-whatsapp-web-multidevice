# Getting Started with WhatsApp Web API

Welcome! This directory contains comprehensive guides to help you get started with the WhatsApp Web API multidevice application.

## Guide Overview

### 1. [Quick Start Guide](quick-start.md)

Get up and running in 5 minutes. Perfect for first-time users and quick evaluation.

**Time required:** 5-10 minutes

### 2. [Installation Guide](installation.md)

Complete installation instructions for all platforms. Covers pre-built binaries, Docker, and building from source.

**Time required:** 15-30 minutes

### 3. [First Message Guide](first-message.md)

Send your first WhatsApp message via API. Learn login methods, message types, and troubleshooting.

**Time required:** 10-15 minutes

### 4. [Configuration Basics](configuration-basics.md)

Essential configuration options including security, webhooks, and database setup.

**Time required:** 20-30 minutes

## Prerequisites

- **Operating System**: Linux, macOS, or Windows
- **WhatsApp Account**: Active phone number with WhatsApp
- **FFmpeg**: Media processing (installation covered in guides)
- **Network**: Internet connection
